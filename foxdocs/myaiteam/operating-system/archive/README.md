# Архив

## Назначение

Эта папка хранит отключённые или заменённые модули. Мы сохраняем их, чтобы:

- **Понимать**, почему что-то было отключено
- **Иметь возможность** вернуть модуль в строй при необходимости
- **Учиться** на том, что не сработало

> Удалять — значит терять контекст. Архивировать — значит сохранять опыт.

## Формат

При отключении модуля:

1. Перенести файлы модуля в эту папку
2. Добавить запись в таблицу ниже с причиной отключения
3. Обновить `MANIFEST.md` — убрать модуль из активных
