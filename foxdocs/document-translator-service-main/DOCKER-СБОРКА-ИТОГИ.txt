⚠️  ВАЖНОЕ ОГРАНИЧЕНИЕ
═════════════════════

Docker на Linux собирает LINUX binary, НЕ Windows .exe!

Почему?
→ PyInstaller собирает для текущей платформы
→ Docker использует Linux контейнер
→ Wine очень сложен и ненадёжен
