# 🌐 Translation Benchmark - Новая папка для тестирования переводчиков

## 📦 Что это?

В директории `/translation_benchmark` создан **независимый тестовый стенд** для сравнения переводчиков с китайского на русский.

## 🎯 Назначение

Сравнение производительности двух библиотек перевода:
1. **Argos Translate** - двухэтапный (zh→en→ru)
2. **M2M100** - прямой (zh→ru)

**Ключевое требование:** Только CPU, без NVIDIA/CUDA!

## 📁 Структура

```
translation_benchmark/
├── benchmark_translators.py    # Основной скрипт
├── run_benchmark.sh            # Автозапуск
├── requirements.txt            # CPU-only зависимости
├── chinese_test_data.txt       # 20 тестовых текстов
├── INDEX.md                    # Навигация
├── QUICK_START.md             # Быстрый старт
├── README.md                   # Полная документация
├── TECHNICAL_NOTES.md         # Технические детали
├── EXAMPLE_OUTPUT.md          # Пример вывода
└── PROJECT_SUMMARY.md         # Сводка проекта
```

## 🚀 Быстрый запуск

```bash
cd translation_benchmark
./run_benchmark.sh
```

## 📊 Что измеряется

- ⏱️ Скорость перевода (общая и на 1 текст)

- 🧠 Использование RAM (средняя и пиковая)
- 📈 Сравнительные таблицы качества переводов
