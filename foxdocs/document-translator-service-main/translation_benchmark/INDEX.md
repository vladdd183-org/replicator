## 📊 Что тестируется?

### 1. Argos Translate
- Путь перевода: **Китайский → Английский → Русский**
- Два языковых пакета: `zh→en` и `en→ru`
- Основан на CTranslate2
- **Преимущества**: Легковесный, меньше RAM
- **Недостатки**: Двухэтапный перевод может терять точность

### 2. M2M100 (facebook/m2m100_418M)
- Путь перевода: **Китайский → Русский** (прямой)
- Многоязычная модель (100 языков)
- 418 миллионов параметров
- **Преимущества**: Прямой перевод, возможно точнее
- **Недостатки**: Требует больше RAM (~1-2 GB)

## 💡 Советы и трюки

### 🚀 Ускорение бенчмарка
```bash
# Использовать все ядра CPU
export OMP_NUM_THREADS=$(nproc)
export MKL_NUM_THREADS=$(nproc)
```

### 💾 Экономия памяти
- Закройте все другие приложения перед запуском
- Увеличьте swap если RAM < 8GB
- Уменьшите количество тестовых текстов

### 🔍 Отладка
```bash
# Проверить использование ресурсов во время работы
htop  # или top

# Проверить доступную память
free -h
```
