# 📦 Сводка проекта: Бенчмарк переводчиков

## 🎯 Цель проекта

Создан **независимый** тестовый стенд для объективного сравнения двух библиотек перевода с китайского на русский:

1. **Argos Translate** - двухэтапный перевод (zh→en→ru)
2. **M2M100** - прямой перевод (zh→ru)

**Ключевое требование:** Только CPU, никаких NVIDIA/CUDA библиотек!

## 🔧 Технические характеристики

### Переводчики

#### 1. Argos Translate
```python
Архитектура:  zh → en → ru (двухэтапный)
Движок:       CTranslate2 (CPU-optimized)
Пакеты:       2 штуки (zh-en, en-ru)
RAM:          ~300-500 MB
Скорость:     ~0.5 сек/текст
CPU:          ~40-60% (1-2 ядра)
```

#### 2. M2M100 (facebook/m2m100_418M) с CTranslate2
```python
Архитектура:  zh → ru (прямой)
Движок:       CTranslate2 с int8 квантизацией (CPU-optimized)
Параметры:    418 миллионов
Размер:       ~250 MB (после квантизации)
RAM:          ~800 MB (меньше благодаря квантизации)
Скорость:     ~0.15-0.3 сек/текст (в 2-4x быстрее PyTorch)
CPU:          ~70-90% (все ядра)
```

### Измеряемые метрики

| Метрика | Описание | Точность |
|---------|----------|----------|
| ⏱️ Время | Время перевода каждого текста | миллисекунды |
| 💻 CPU | Загрузка процессора | % на ядро |
| 🧠 RAM | Использование памяти | мегабайты |
| 📊 Агрегаты | Среднее, пиковое, общее | вычисляется |

## 🔍 Детали реализации

### Классы и структуры данных

```python
# Мониторинг
class ResourceMonitor:
    - Отслеживание CPU/RAM через psutil
    - Baseline память для точных измерений

# Переводчики
class ArgosTranslateWrapper:
    - Инициализация пакетов zh-en, en-ru
    - Двухэтапный перевод
    - Мониторинг для каждого перевода

class M2M100Wrapper:
    - Загрузка модели facebook/m2m100_418M
    - Принудительный CPU режим
    - Оптимизация через torch.no_grad()

# Результаты
@dataclass TranslationResult:
    - original: str
    - translated: str
    - time_seconds: float
    - cpu_percent: float
    - memory_mb: float

@dataclass BenchmarkResult:
    - translator_name: str
    - results: List[TranslationResult]
    - total_time: float
    - avg_time_per_text: float
    - avg_cpu_percent: float
    - avg_memory_mb: float
    - peak_memory_mb: float
```
