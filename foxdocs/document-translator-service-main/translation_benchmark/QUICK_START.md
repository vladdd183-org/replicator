# 🚀 Быстрый старт

## Минимальная инструкция для запуска бенчмарка

### Вариант 1: Автоматический запуск (рекомендуется)

```bash
cd translation_benchmark
./run_benchmark.sh
```

Скрипт автоматически:
- ✅ Создаст виртуальное окружение
- ✅ Установит все зависимости (CPU-only)
- ✅ Запустит бенчмарк

### Вариант 2: Ручная установка

```bash
cd translation_benchmark

# 1. Создать виртуальное окружение
python3 -m venv venv
source venv/bin/activate

# 2. Установить PyTorch (CPU-only)
pip install torch --index-url https://download.pytorch.org/whl/cpu

# 3. Установить остальные зависимости
pip install -r requirements.txt

# 4. Запустить бенчмарк
python benchmark_translators.py
```

## ⏱️ Время выполнения

- **Первый запуск**: 10-15 минут (загрузка моделей ~1-2 GB)
- **Последующие запуски**: 2-5 минут (модели уже скачаны)

## 📊 Что вы увидите

Бенчмарк выведет три таблицы:

1. **Таблица переводов** - сравнение качества переводов
2. **Метрики производительности** - время, CPU, RAM
3. **Выводы** - кто быстрее и эффективнее

## 🔧 Системные требования

- Python 3.11+
- 4-8 GB RAM (свободной)
- 2-3 GB места на диске
- Стабильное интернет-соединение (для первой загрузки)
