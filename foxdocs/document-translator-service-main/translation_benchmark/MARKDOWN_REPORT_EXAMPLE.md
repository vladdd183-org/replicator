# 📊 Пример структуры Markdown отчёта

Этот файл показывает структуру автоматически генерируемого отчёта после запуска бенчмарка.

## 💻 Информация о системе

- **ОС:** Linux 6.12.40
- **Процессор:** x86_64
- **Архитектура:** 64bit
- **Python:** 3.12.x
- **Количество тестовых фраз:** 20

## 🌐 Протестированные переводчики

1. **T5-CT2 (T5-Small)** - Прямой перевод zh→ru через T5 с CTranslate2 (int8)
2. **T5-CT2 (T5-Base)** - Прямой перевод zh→ru через T5 с CTranslate2 (int8)
3. **M2M100-ONNX** - Прямой перевод zh→ru через ONNX Runtime (q4f16)
4. **M2M100-CT2** - Прямой перевод zh→ru через CTranslate2 (int8)
5. **Argos Translate** - Двухэтапный перевод zh→en→ru через CTranslate2

## 📈 Сводная статистика производительности

| Переводчик | Общее время | Время/текст | Средний CPU | Средняя RAM | Пиковая RAM |
|------------|-------------|-------------|-------------|-------------|-------------|
| **T5-Small-CT2** | 3.45s | 0.173s | 65.2% | 245.3 MB | 289.1 MB |
| **T5-Base-CT2** | 5.82s | 0.291s | 72.8% | 534.7 MB | 598.2 MB |
| **M2M100-ONNX** | 4.21s | 0.211s | 58.3% | 456.2 MB | 512.8 MB |
| **M2M100-CT2** | 6.15s | 0.308s | 81.5% | 824.5 MB | 956.2 MB |
| **Argos Translate** | 9.87s | 0.494s | 45.2% | 312.3 MB | 378.9 MB |

## 🏆 Выводы

### ⚡ Самый быстрый: T5-Small-CT2

- **Общее время:** 3.45s
- **Среднее время на текст:** 0.173s

**Сравнение скорости:**

- M2M100-ONNX: в **1.22x** раз медленнее
- T5-Base-CT2: в **1.69x** раз медленнее
- M2M100-CT2: в **1.78x** раз медленнее
- Argos Translate: в **2.86x** раз медленнее

### 🧠 Самый экономичный по памяти: T5-Small-CT2

- **Пиковая память:** 289.1 MB
- **Средняя память:** 245.3 MB

**Сравнение использования памяти:**

- Argos Translate: на **89.8 MB** больше
- M2M100-ONNX: на **223.7 MB** больше
- T5-Base-CT2: на **309.1 MB** больше
- M2M100-CT2: на **667.1 MB** больше

## 💡 Рекомендации

### Для максимальной скорости:
Используйте **T5-Small-CT2** (0.173s на текст)

### Для минимального использования памяти:
Используйте **T5-Small-CT2** (289.1 MB)
