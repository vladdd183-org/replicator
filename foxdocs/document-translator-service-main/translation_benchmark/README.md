# 🌐 Бенчмарк переводчиков: Китайский → Русский (CPU ONLY)

Тестовый стенд для сравнения производительности **5 различных** реализаций перевода:
1. **T5-Small + CTranslate2** (прямой перевод zh → ru) - компактная модель
2. **T5-Base + CTranslate2** (прямой перевод zh → ru) - точная модель
3. **M2M100 + ONNX Runtime** (прямой перевод zh → ru) с готовыми квантизованными моделями
4. **M2M100 + CTranslate2** (прямой перевод zh → ru) с int8 квантизацией
5. **Argos Translate** (zh → en → ru через промежуточный английский) + CTranslate2

## ⚠️ Важно: Только CPU с CTranslate2!

Этот бенчмарк использует **ТОЛЬКО CPU**, без каких-либо NVIDIA/CUDA библиотек!

**Новое в этой версии:**
- ✅ **5 переводчиков** для полного сравнения
- ✅ **T5 модели** (Small и Base) - специализированы для ru-zh-en пар
- ✅ **ONNX Runtime** - максимальная оптимизация для CPU с готовыми квантизованными моделями
- ✅ **CTranslate2** с int8 квантизацией - в 2-4 раза быстрее обычного PyTorch
- ✅ **Автоматическое сохранение** результатов в красиво оформленный Markdown файл
- ✅ Автоматическая конвертация моделей при первом запуске
- ✅ Минимальное использование памяти благодаря квантизации
- ✅ Детальные отчёты с графиками и рекомендациями

## 📋 Требования

- Python 3.11+
- Минимум 4GB RAM (рекомендуется 8GB+)
- CPU с поддержкой AVX2 (современные процессоры)

## 🚀 Установка

### Шаг 1: Создать виртуальное окружение

```bash
cd translation_benchmark
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

### Шаг 2: Установить зависимости

```bash
# ВАЖНО: Убедитесь что не установлены CUDA версии
pip install --upgrade pip

# Установка PyTorch CPU-only версии
pip install torch --index-url https://download.pytorch.org/whl/cpu

# Установка остальных зависимостей
pip install -r requirements.txt
```

### Шаг 3: Проверка установки (без CUDA)

```bash
python -c "import torch; print('CUDA доступна:', torch.cuda.is_available())"
# Должно вывести: CUDA доступна: False
```

## 🎯 Использование

### Запуск бенчмарка

```bash
python benchmark_translators.py
```

Скрипт автоматически:
1. ✅ Проверит наличие необходимых библиотек
2. 📦 Загрузит и установит языковые пакеты для Argos Translate
3. 📥 Загрузит модели T5 и M2M100 (при первом запуске ~1-3 GB)
4. 🔄 Конвертирует модели в CTranslate2 формат (только при первом запуске)
5. 🚀 Запустит бенчмарк на всех текстах из `chinese_test_data.txt`
6. 📊 Выведет детальную сравнительную таблицу в терминал
7. 💾 Сохранит полный отчёт в Markdown файл

### Вывод результатов

#### В терминале:
Бенчмарк выводит сравнительные таблицы для всех переводчиков с метриками производительности.

#### В Markdown файле:
После завершения бенчмарка создаётся файл `benchmark_results_YYYYMMDD_HHMMSS.md` со следующими разделами:

1. **💻 Информация о системе** - ОС, процессор, архитектура
2. **🌐 Список переводчиков** - все протестированные модели
3. **📈 Сводная статистика** - таблица с метриками производительности
4. **🏆 Выводы** - автоматический анализ:
   - ⚡ Самый быстрый переводчик
   - 🧠 Самый экономичный по памяти
   - Сравнение всех переводчиков
5. **💡 Рекомендации** - что выбрать для разных сценариев
6. **📝 Детальные результаты** - каждый текст с переводами от всех систем
7. **📊 ASCII визуализация** - графики времени и памяти
8. **🔧 Технические детали** - полная статистика каждого переводчика

## 📊 Что измеряется

Для каждого переводчика:
- ⏱️ **Общее время** выполнения всех переводов
- 📈 **Время на 1 единицу** перевода (средний показатель)
- 💻 **Средняя загрузка CPU** в процентах
- 🧠 **Средняя загрузка RAM** в мегабайтах
- 📊 **Пиковая загрузка RAM** (максимальное значение)

## 🔧 Настройка тестовых данных

Вы можете заменить содержимое `chinese_test_data.txt` своими текстами:
- Один текст на строку
- Кодировка UTF-8
- Пустые строки игнорируются

## 🐛 Решение проблем

### Ошибка: CUDA библиотеки найдены

Если скрипт обнаруживает CUDA, убедитесь что:
```bash
# Проверьте что PyTorch установлен без CUDA
pip uninstall torch
pip install torch --index-url https://download.pytorch.org/whl/cpu
```

### Ошибка: Недостаточно памяти

M2M100 требует ~1-2 GB RAM. Если памяти мало:
- Закройте другие приложения
- Уменьшите количество текстов в `chinese_test_data.txt`
- Используйте меньшую модель (можно попробовать экспериментальные версии)

### Ошибка: Argos пакеты не загружаются

```bash
# Вручную обновите индекс пакетов
python -c "import argostranslate.package; argostranslate.package.update_package_index()"
```

## 📚 Технические детали

### T5-Small (utrobinmv/t5_translate_en_ru_zh_small_1024)
- Прямой перевод `zh→ru` без промежуточного языка
- Компактная модель (~60M параметров)
- Специализирована для ru-zh-en пар
- CTranslate2 с int8 квантизацией
- Минимальное использование памяти (~200-300 MB)
- Быстрая скорость перевода

### T5-Base (utrobinmv/t5_translate_en_ru_zh_base_200)
- Прямой перевод `zh→ru` без промежуточного языка
- Более крупная модель (~220M параметров)
- Специализирована для ru-zh-en пар
- CTranslate2 с int8 квантизацией
- Лучшее качество перевода
- Умеренное использование памяти (~500-600 MB)

### M2M100-ONNX (418M)
- Прямой перевод `zh→ru` без промежуточного языка
- ONNX Runtime с оптимизациями для CPU
- Готовые квантизованные модели (q4f16)
- Поддерживает 100 языков
- Отличная скорость на CPU

### M2M100-CT2 (418M)
- Прямой перевод `zh→ru` без промежуточного языка
- CTranslate2 с int8 квантизацией
- Поддерживает 100 языков и 9900 направлений перевода
- Универсальная multilingual модель

### Argos Translate
- Использует два языковых пакета: `zh→en` и `en→ru`
- Перевод происходит в два этапа через английский язык
- Основан на CTranslate2 (CPU-optimized)
- Меньше требований к памяти
