Что делает pyclipper:
--------------------
- Работа с полигонами (многоугольниками)
- Используется PaddleOCR для детекции границ текста
- Необходим для функции автоматического поиска текста
- Без него работают только ручные выделения областей

Почему это последняя зависимость:
--------------------------------
- Все основные зависимости уже установлены
- Runtime hook патчит проверки зависимостей
- pyclipper - это utility библиотека, импортируемая динамически
- После добавления pyclipper система должна быть полностью функциональна

 💡 СОВЕТЫ ПО ИСПОЛЬЗОВАНИЮ:

1. Первый запуск может быть долгим (скачивание моделей OCR)

2. При обработке больших документов используйте выделение областей
   - Быстрее чем автоматическая детекция

3. Если модели не загрузились - проверьте интернет соединение
   - Модели скачиваются с серверов PaddlePaddle
   - Размер моделей: ~500MB-1GB

4. Логи полезны для диагностики
   - Все действия логируются через logfire
   - При ошибках смотрите полный traceback
