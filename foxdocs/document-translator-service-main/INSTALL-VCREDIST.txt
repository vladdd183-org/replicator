===================================================================
ВАЖНО: Visual C++ Redistributable ОБЯЗАТЕЛЕН!
===================================================================

Если приложение не запускается с ошибкой DLL, установите:

Visual C++ Redistributable 2015-2022 (x64)
Ссылка: https://aka.ms/vs/17/release/vc_redist.x64.exe

===================================================================
ИНСТРУКЦИЯ ПО УСТАНОВКЕ:
===================================================================

1. Скачайте файл по ссылке выше
2. Запустите vc_redist.x64.exe
3. Следуйте инструкциям установщика
4. Перезагрузите компьютер (если попросит)
5. Запустите DocumentTranslator.exe

===================================================================
ПРИЗНАКИ ПРОБЛЕМЫ:
===================================================================

Если вы видите ошибки типа:
- "The specified module could not be found"
- "DLL load failed"
- "c10.dll" not found
- "_sentencepiece" import error

Это означает что VC++ Redistributable не установлен!

===================================================================
ПРОВЕРКА УСТАНОВКИ:
===================================================================

После установки VC++ Redistributable:
1. Откройте "Панель управления"
2. "Программы и компоненты"
3. Найдите "Microsoft Visual C++ 2015-2022 Redistributable (x64)"
4. Если есть - всё OK!
