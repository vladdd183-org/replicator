# /slang — Словарь разработчика

Работа со словарём сленга, жаргона и мемов.

## Источник данных

Загрузи словарь: `agent-os/slang/dictionary.md`

## Обработка запроса

| Запрос | Действие |
|--------|----------|
| `/slang` | Показать справку по командам |
| `/slang <слово>` | Найти слово в словаре |
| `/slang dev` | Показать dev-сленг |
| `/slang python` | Показать Python-сленг |
| `/slang porto` | Показать Hyper-Porto сленг |
| `/slang memes` | Показать мемы |
| `/slang commands` | Показать команды → действия |
| `/slang review` | Показать фразы code review |

## Пример

```
Пользователь: /slang экшн
→ Найди "экшн" в словаре и покажи значение
```
