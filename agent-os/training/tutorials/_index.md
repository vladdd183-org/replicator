> Пошаговые руководства для изучения архитектуры

## Обзор

Туториалы — это структурированные пошаговые руководства. Каждый туториал фокусируется на конкретной теме и включает:

- Теоретическое введение
- Практические шаги
- Примеры кода
- Проверочные задания

## Как проходить туториал

### 1. Подготовка

```bash
# Убедись, что проект запускается
cd /home/vladdd183/Рабочий\ стол/kwork/new_porto
uv run python -m src.Main
```

### 2. Начало туториала

Скажи AI-агенту:

```
@training туториал 01-first-action
```

Или просто открой файл туториала и следуй инструкциям.

### 3. Выполнение шагов

Каждый туториал разбит на шаги. Выполняй их последовательно:

```markdown
## Шаг 1: Создание файла

Создай файл `src/Containers/AppSection/TrainingModule/Actions/MyAction.py`
```

### 4. Проверка

После каждого шага можешь проверить результат:

```
@training проверь шаг 1
```

### 5. Завершение

В конце туториала — итоговое задание для закрепления.

## Создание нового туториала

### Шаблон

```markdown
# {Номер}. {Название}

> {Краткое описание}

## Цель

{Что научишься делать}

## Требования

- {Пререквизит 1}
- {Пререквизит 2}

## Шаг 1: {Название шага}

{Инструкции}

## Шаг 2: {Название шага}

{Инструкции}

...

## Итоговое задание

{Задание для самостоятельного выполнения}

## Что дальше?

{Ссылки на следующие туториалы}
```

### Правила

1. Один туториал — одна тема
2. Шаги должны быть атомарными
3. Код должен быть проверяемым
4. Включай объяснения "почему", а не только "как"
