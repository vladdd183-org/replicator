---

## 📁 Содержимое

| Файл | Описание |
|------|----------|
| `dictionary.md` | Полный словарь сленга |

---

## 🎯 Зачем это нужно

1. **AI понимает сленг** — "накидай апишку" = создай REST endpoint
2. **Быстрая коммуникация** — не нужно формулировать формально
3. **Единый язык** — команда говорит одинаково

## 🎮 Команда

```

/slang list           — все категории
/slang porto          — Hyper-Porto специфичный
```

---

## 🔄 Как AI использует

**Вход:** "Запили экшн, который дёргает репу и пуляет евент"

**AI понимает:**
- запили = создай
- экшн = Action (Use Case)
- дёргает = вызывает
- репу = Repository
- пуляет = публикует
- евент = Domain Event

**AI делает:**
1. Создаёт Action
2. Инжектит Repository через UoW
3. Добавляет публикацию Event

## ➕ Добавление сленга

Если встретил новый сленг:
1. Определи значение из контекста
2. Добавь в `dictionary.md` в соответствующую категорию
3. Используй в ответах (опционально)
