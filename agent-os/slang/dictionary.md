## 🆘 AI: Как использовать

Когда пользователь говорит сленгом, AI должен:

1. **Понять** что имеется в виду
2. **Выполнить** соответствующее действие
3. **Ответить** в том же стиле (опционально)

### Примеры

**Запрос:** "Накидай экшн для создания ордера, чтобы пулил евент"

**Перевод:** Создать `CreateOrderAction` с публикацией `OrderCreated` event через UoW.

---

**Запрос:** "Там репа течёт, почекай"

**Перевод:** Проверить Repository на memory leaks.

---

**Запрос:** "Зарефактори эту лапшу, выгляит как говнокод"

**Перевод:** Улучшить структуру кода, убрать сложность.
