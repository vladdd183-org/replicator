# Standards Evolution Proposals

Система предложений по эволюции стандартов.

## Статусы

- `pending/` — Ожидают review
- `applied/` — Применены
- `rejected/` — Отклонены

## Формат proposal

```yaml
---
id: PROPOSAL-YYYY-MM-DD-NNN
type: new_pattern | update | deprecation
confidence: 0.0-1.0
evidence_count: N
---
```
