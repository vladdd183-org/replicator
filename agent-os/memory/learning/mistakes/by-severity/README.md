# Mistakes by Severity

Ошибки, сгруппированные по уровню серьёзности.

## Уровни

### Critical
Ошибки, приводящие к потере данных, безопасности или полной неработоспособности.

### Major  
Ошибки, значительно влияющие на функциональность или производительность.

### Minor
Мелкие ошибки, стилистические проблемы, неоптимальные решения.

## Именование файлов

`YYYY-MM-DD-severity-short-name.md`
