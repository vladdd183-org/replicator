# Mistakes — Каталог ошибок

Документация ошибок и способов их предотвращения.

## Подпапки

- `by-severity/` — ошибки по уровню серьёзности (critical, major, minor)

## Формат файла

```markdown
# Mistake: [Название]

## Что произошло
[Описание ошибки]

## Почему произошло
[Root cause]

## Как исправили
[Решение]

## Как предотвратить
[Превентивные меры]

## Severity
critical | major | minor
```
