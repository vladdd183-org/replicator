# Insights — Инсайты

Извлечённые инсайты и наблюдения из опыта работы.

## Формат файла

```markdown
# Insight: [Название]

## Наблюдение
[Что было замечено]

## Вывод
[Какой вывод сделан]

## Применение
[Как использовать этот инсайт]

## Дата
YYYY-MM-DD
```

## Категории

- `architecture-*.md` — архитектурные инсайты
- `debugging-*.md` — отладка
- `performance-*.md` — производительность
- `dx-*.md` — developer experience
