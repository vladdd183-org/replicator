# Experimental Patterns — Экспериментальные паттерны

Паттерны в стадии эксперимента. Требуют проверки перед широким использованием.

## Статусы

- **Draft** — идея, требует проработки
- **Testing** — в процессе тестирования
- **Review** — готов к ревью для перехода в verified

## Формат файла

```markdown
# Experimental: [Название]

## Status
Draft | Testing | Review

## Идея
[Описание идеи]

## Гипотеза
[Что проверяем]

## Текущий статус
[Прогресс]

## Следующие шаги
[Что нужно сделать]
```
