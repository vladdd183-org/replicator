# ADR — Architecture Decision Records

Записи архитектурных решений по формату ADR.

## Формат файла

```markdown
# ADR-XXX: [Название решения]

## Status
Proposed | Accepted | Deprecated | Superseded by ADR-YYY

## Context
[Контекст и проблема]

## Decision
[Принятое решение]

## Consequences
[Последствия решения]

### Positive
- [Плюсы]

### Negative
- [Минусы]

## Date
YYYY-MM-DD
```

## Именование

`ADR-XXX-short-name.md` (например: `ADR-001-use-result-pattern.md`)
