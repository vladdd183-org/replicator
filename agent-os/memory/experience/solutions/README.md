# Solutions — Успешные решения

Каталог проверенных решений для повторного использования.

## Формат файла

```markdown
# Solution: [Название]

## Проблема
[Описание проблемы]

## Решение
[Код или описание решения]

## Контекст применения
[Когда использовать это решение]

## Связанные модули
[Список модулей]
```

## Именование

`category-problem-name.md` (например: `auth-jwt-refresh.md`)
