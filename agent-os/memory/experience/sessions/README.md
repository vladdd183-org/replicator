# Sessions — Логи сессий

Записи о выполненных задачах и сессиях работы.

## Формат файла

```markdown
# Session: YYYY-MM-DD-HH-MM

## Задача
[Описание задачи]

## Решение
[Краткое описание решения]

## Результат
- [ ] Успех / Частичный успех / Неудача

## Заметки
[Дополнительные наблюдения]
```

## Именование

`YYYY-MM-DD-task-name.md`
