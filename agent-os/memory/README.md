# Agent Memory System

Централизованное хранилище памяти AI-агента для накопления опыта и знаний.

## Структура

```
memory/
├── experience/     # Накопленный опыт работы
│   ├── sessions/   # Логи сессий и решённых задач
│   └── solutions/  # Успешные решения для повторного использования
├── learning/       # Обучение на ошибках
│   ├── insights/   # Извлечённые инсайты
│   └── mistakes/   # Каталог ошибок и их исправлений
├── knowledge/      # База знаний
│   ├── architecture/  # Архитектурные решения (ADR)
│   └── patterns/      # Паттерны (verified/experimental)
└── context/        # Контекст проекта
    └── modules/    # Информация о модулях
```

## Принципы

1. **Append-only** — новые записи добавляются, старые не удаляются
2. **Structured** — каждая запись следует формату
3. **Searchable** — записи легко искать по ключевым словам
4. **Actionable** — записи содержат конкретные действия
