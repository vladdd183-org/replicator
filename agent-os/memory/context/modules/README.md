# Modules — Информация о модулях

Контекстная информация о каждом модуле проекта.

## Формат файла

```markdown
# Module: [Название]

## Назначение
[Краткое описание]

## Ключевые компоненты
- Actions: [список]
- Tasks: [список]
- Events: [список]

## Зависимости
- Входящие: [кто использует этот модуль]
- Исходящие: [от кого зависит]

## Особенности
[Специфика модуля]

## Последние изменения
[Краткая история]
```

## Именование

`ModuleName.md` (например: `UserModule.md`)
