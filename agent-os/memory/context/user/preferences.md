## 🎨 Стиль кода

### Именование
- **Язык:** Английский для кода, русский для комментариев
- **Переменные:** snake_case
- **Классы:** PascalCase
- **Константы:** UPPER_SNAKE_CASE

### Форматирование
- **Line length:** 88 (ruff default)
- **Quotes:** Double quotes
- **Trailing comma:** Yes

### Type hints
- **Строгость:** Strict (все функции типизированы)
- **Optional:** Использовать `T | None` вместо `Optional[T]`

## 📝 Комментарии

- **Язык:** Русский
- **Docstrings:** Google style
- **Когда:** Для сложной логики, не для очевидного кода

## 🏗️ Архитектура

### Предпочтения
- **Явность** над магией
- **Композиция** над наследованием
- **Immutability** где возможно
- **Railway-Oriented Programming** для error handling

### Что НЕ любит
- Относительные импорты
- Exceptions для бизнес-ошибок
- Магические декораторы без явной регистрации
- Service Locator
