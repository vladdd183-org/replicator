# /slang — Справка

## Что это?

Словарь сленга помогает AI понимать неформальную речь разработчиков.

**Пример:**
```
Ты: "Накидай экшн, который дёргает репу и пуляет евент"

AI понимает:
- накидай → создай
- экшн → Action
- дёргает → вызывает
- репу → Repository  
- пуляет → публикует
- евент → Event

AI делает: Создаёт Action с Repository и Event
```

# /slang <слово> — Поиск

## Примеры

```
/slang экшн
→ Action (Use Case в Hyper-Porto)
→ Пример: "Напиши экшн для регистрации"

/slang костыль
→ Временное/плохое решение
→ Пример: "Это костыль, потом поправим"

/slang дишка
→ Dishka (DI framework)
→ Пример: "Инжектим через дишку"
```

# /slang commands — Команды → Действия

| Говорят | Делай |
|---------|-------|
| "накидай апишку" | Создай REST endpoint |
| "запили экшн" | Создай Action |
| "вынеси в таску" | Создай Task |
| "добавь в репу" | Добавь метод в Repository |
| "зарегай в провайдере" | Зарегистрируй в Providers.py |
| "верни резалт" | Return Result[T, E] |
| "заверни в саксесс" | Return Success(value) |
| "пульни евент" | Опубликуй Event через UoW |
| "накати миграцию" | Примени DB migration |
| "выкати на прод" | Задеплой на production |
| "пофикси линтер" | Исправь ошибки ruff/mypy |

# /slang review — Фразы Code Review

| Фраза | Значение |
|-------|----------|
| "Не питоник" | Не Python-style |
| "Не рейзь, верни фейлюр" | Используй Result |
| "Почему относительный?" | Нужен абсолютный импорт |
| "Откуда хардкод?" | Вынеси в конфиг |
| "Где тайпхинты?" | Добавь типизацию |
| "Много ответственности" | Разбей на части |
| "Оверинжениринг" | Упрости |
| "Велосипед" | Используй либу |
