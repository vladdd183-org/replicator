## 🔗 Структура команды

Каждая команда содержит:
- **Синтаксис** — как вызывать
- **Параметры** — что передавать
- **Примеры** — готовые к использованию
- **Что создаётся** — полный код/структура
- **Действия после** — что сделать дальше
