# Feedback Responses

Хранилище ответов на собранный фидбек.

## Назначение

Эта директория содержит:
- Ответы на отзывы пользователей
- Результаты обработки фидбека
- Статусы рассмотрения предложений
- История коммуникации по фидбеку

## Структура

```
responses/
├── pending/         # Ожидающие ответа
├── resolved/        # Обработанные
└── archived/        # Архивные
```

## Формат файлов

Каждый файл ответа содержит:
- ID исходного фидбека
- Дата ответа
- Текст ответа
- Предпринятые действия
- Статус (pending/resolved/rejected)

## Процесс обработки

1. Фидбек поступает в collection/
2. Создается запись в responses/pending/
3. После обработки перемещается в resolved/
4. Периодически архивируется в archived/
